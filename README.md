# ProgettoMobility (July 2021)

University project for the "Web & Mobile Programming" and the "Software Engineering" courses (BS in Computer Engineering, Università degli Studi di Palermo @unipa)

Contributors: Vincenzo Messina, Giorgio Tocco, Roberto Graffeo, Marco Costanzo

## General Description
ProgettoMobility is a vehicle rental web and mobile application with a user-friendly interface for managing reservations.

## Technologies Used
- Back-end: NodeJS and Express
- Front-end: HTML, CSS (Bootstrap) and JavaScript.
- MySQL DBMS - Manages relational database storage.
